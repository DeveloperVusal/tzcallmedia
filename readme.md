## Краткое руководство
### Запустить сборку
```console
bash docker
```

> Важно! Необходимо сначала запустить очередь, после сборки.

### Очередь RabbitMQ и запись в MariaDB
```console
bash exec-queue
```

### Отправка записей в очередь RabbitMQ
```console
bash exec-sent
```

### Получение данных
```console
bash exec-select
```
